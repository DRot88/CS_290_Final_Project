Fizz Buzz

In this section, we will discuss an extremely common interview question, known as "Fizz Buzz."

The general idea of this is to develop an algorithm that counts from 1 to some arbitrarily chosen number. The caveat being that every number divisible by 3 is replaced with the word Fizz, every number divisibile by 5 is replaced with the word Buzz and every number divisible by both 3 and 5 replaced with the word FizzBuzz.

The answer here is show in JavaScript.

var user_input = prompt("How many items would you like to see for the FizzBuzz game?");
var FizzBuzz = [];
var x = 1;
while (x <= user_input) {
  if ((x % 3 == 0) && (x % 5 == 0)) {
    FizzBuzz.push("FizzBuzz");
  } else if (x % 3 == 0) {
    FizzBuzz.push("Fizz");
  } else if (x % 5 == 0) {
    FizzBuzz.push("Buzz");
  } else {
    FizzBuzz.push(x);
  }
  x += 1;
}

Fibonacci Sequence

This program will prompt the user for a number, n, which will be used as a limit for the amount of iterations in the fibonacci sequence. The sequence of numbers will be printed to the console, ending after the nth iteration.

The fibonacci sequnece is a series of numbers where each new number in the sequence is equal to the sum of the prior two numbers in the series.



The answer here is show in JavaScript. 

var num1 = 0;
var num2 = 1;
var num3;
 
var limit = prompt("Enter the maximum fibonacci iterations you would like to see");
console.log(num1);
console.log(num2);

for(var i=3; i <= limit; i++)
{
  num3 = num1 + num2;
  num1 = num2;
  num2 = num3;

  console.log(num3);
}
